HMFcalc: An online Halo Mass Function calculator

{% endblock %} {% block content %}

This is a web-application for calculating the Halo Mass Function. It is written by Steven Murray with others at the International Centre for Radio Astronomy Research (ICRAR).

If you would like to acknowledge use of this application, please cite this article.

Please check out other information about the mass function and this calculator by following the links at the top of this page.

Note that this calculator is under active development. We are confident it does what it advertises well and robustly, but there is always room for improvement. Please contact us directly or let us know via the Twitter panel on the right if you find any bugs, have ideas for new features, or notice any design flaws! If you notice something after producing a plot, don't hesitate to click the "Contact Us!" link at the top of the page -- your data will be persistent!

Finally, the calculation backend for this web-app is written in Python and is made open-source via github. If you have a working python installation you can install it simply with "pip install hmf".
