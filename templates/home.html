This is an app for calculating the Halo Mass Function, written by Steven Murray at the International Centre for Radio Astronomy Research (ICRAR).

Please acknowledge use of this application, by citing Murray, Power & Robotham (2013) .

If you find bugs or issues, or have ideas for improvements, the best way to report it is to file an issue directly. Otherwise, you can contact us directly.

The backend for this app is written in Python. If you have a working python installation you can install it simply with pip install hmf.
